# Ashish Vail — Portfolio Website

> **Professional Web Developer** specializing in custom tools, plugins, and high-quality websites.

A clean, responsive, single-page portfolio built with pure HTML, CSS, and vanilla JavaScript — no framework dependencies, fast by default.

## Features

- **Hero section** — animated introduction with name, title, and call-to-action buttons
- **About section** — brief personal bio
- **Skills section** — six skill cards (Front-End, Back-End, Custom Tools & Plugins, Performance/SEO, DevOps, UI/UX)
- **Projects section** — six featured project cards highlighting custom tools, browser extensions, and high-quality websites
- **Contact form** — client-side validated form with accessible error messages
- Sticky, blur-backdrop navigation with active-section highlighting
- Scroll-reveal animations (respects `prefers-reduced-motion`)
- Fully responsive, mobile-friendly layout

## File Structure

```
ashish-portfolio/
├── index.html   # Page markup
├── styles.css   # Design tokens, layout, components
├── script.js    # Navigation, animations, form validation
└── README.md
```

## Development

Open `index.html` directly in a browser — no build step required.

For a local dev server with live reload, run:

```bash
npx serve .
# or
python3 -m http.server
```
